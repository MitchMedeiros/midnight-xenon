Vim Motions Default Keybinds

| - start of line
$ - end of line
w - start of next word
e - end of the next word
b - start of previous word
f + <char> - jump to first occurrence of the character on the line
; - jump to the next occurrence of the character
i - insert after character
a - insert before character
y - yank
d - delete modified
x - delete character at cursor
g + ; - last edit


Larger Navigate
ctl - f - down full page
ctl - b - up full page
ctl - d - down half page
ctl - u - up half page
